# Labor Connect Gujarat

A data-driven web portal built with Python and Django to bridge the gap between local skilled workers (Masons, Electricians, Painters) and potential employers across Gujarat.

## Features
- **Worker Directory:** Organized listings for skilled workers across Ahmedabad, Rajkot, Bhavnagar, and Porbandar.
- **Admin Control:** Secure Django Admin panel for managing worker profiles and verifying data.
- **Dynamic Logic:** Automated "Premium" status filtering and badge rendering for worker profiles.
- **Responsive UI:** Clean interface for cross-device accessibility.

## Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML, CSS
- **Database:** SQLite
